# Firebase-версия

- Supabase полностью удалён.
- Добавлен Firebase Authentication.
- Добавлен Cloud Firestore.
- Вход только по email и паролю.
- Письма и Magic Link не используются.
- Смена пароля с повторной аутентификацией текущим паролем.
- Команды и участники сохраняются в Firestore.
- Сессии, задачи, раунды и результаты сохраняются в Firestore.
- Значения голосов скрыты до раскрытия.
- Факт голосования виден всей команде.
- Добавлены realtime-обновления.
- Добавлен постоянный локальный кэш.
- Добавлено каскадное удаление вложенных данных.
- Системные alert для сетевых ошибок не используются.
